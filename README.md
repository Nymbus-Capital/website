# Nymbus Capital Website

Modern website for Nymbus Capital — Canada's AI-centric bond manager.

## Tech Stack
- **Next.js 15** (App Router, TypeScript)
- **Tailwind CSS v4** 
- **Framer Motion** + **GSAP** for animations
- **D3.js** + **Chart.js** for data visualization
- **Aceternity UI** inspired components (Background Beams, Reactive Grid, Typewriter, Moving Border)

## Development
```bash
npm install
npm run dev
```

## Build & Deploy
```bash
npm run build
```
Static export goes to `./out/` and is auto-deployed to GitHub Pages via Actions.

## Design System
- All-white theme: #F8FAFC (recessed), #FAFAFA (canvas), #FFFFFF (elevated)
- Blue accent: #4285F4
- Font: Poppins
- Animation intensity: 10/10
